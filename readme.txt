 Crea e stampa i documenti e richiedi il ritiro delle tue spedizioni.

== Description ==

Integra e velocizza la creazione delle spedizioni dai tuoi ordini WooCommerce.
Crea e stampa i documenti e richiedi il ritiro.
Il plugin è integrato con WooCommerce ma può essere usato liberamente come semplice plugin WordPress.

== Installation ==

= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

== Frequently Asked Questions ==

= Cosa devo fare per poter usare il servizio? =

Registrati gratuitamente sul sito MyEXPRESS.it e utilizza l'email e la password che hai scelto.

= Il plugin gestisce i ritiri e la creazione dei documenti? =

Certamente, dopo aver completato il pagamento il plugin crea sia i documenti di trasporto da applicare sul singolo pacco, che un documento di riepilogo da far firmare all'incaricato del ritiro.
Nel carrello potrai scegliere la data del ritiro che preferisci.

= Come pago le mie spedizioni? =

Sul sito internet MyEXPRESS.it potrai acquistare i crediti per effettuare le tue spedizioni. Potrai decidere se pagare con Paypal o Bonifico bancario.

= Dove posso spedire? =

Puoi spedire in tutta l'Unione Europea.

== Screenshots ==

1. Richiama velocemente i dati di spedizione di una ordine cliccando sull'icona nella lista ordini Woocommerce
2. I campi si autocompleteranno (Mittende con i dati inseriti nelle impostazioni e destinatario con i dati richiamati dall'ordine)
3. Puoi aggiugnere una spedizione anche senza aggiungerla da Woocommerce 
4. Nel carrello seleziona la data ritiro desiderata e procedi con il pagamento e la contestuale creazione dei documenti
5. Avrai a disposizione lo storico e le informazioni di tutte le spedizioni create
6. Potrai anche ricercare le spedizioni in base a paramentri specifici
7. Gestisci le impostazioni per velocizzare l'autocompletamento dei form nella fase di creazione della spedizione
